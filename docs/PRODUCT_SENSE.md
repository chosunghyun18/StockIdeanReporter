# Product Sense

## 핵심 사용자 가치

**"분석은 AI가, 판단은 사람이"**

복잡한 재무/기술적 분석을 자동화하되, 최종 투자 판단은 사용자에게 남긴다.
시스템은 아이디어를 제공하고, 인간이 리스크를 결정한다.

## 사용자 페르소나

| 페르소나 | 특징 | 주요 니즈 |
|----------|------|-----------|
| 개인 투자자 | 정보 비대칭 해소 원함 | 빠른 종목 리서치 |
| 소규모 펀드 운용 | 종목 스크리닝 자동화 | 반복 작업 절감 |

## 성공 지표

- 분석 1건 소요 시간 < 3분
- 투자 아이디어 신뢰도 ≥ 65% 시 Slack 전송
- 주간 분석 종목 수 ≥ 10개

## 절대 하지 말아야 할 것

- 100% 자동 주문 집행 (판단은 인간이)
- 신뢰도 < 55% 아이디어 자동 전송
- 환경변수 없이 하드코딩된 API 키 사용

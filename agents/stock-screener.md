# stock-screener

스윙 매매 관점에서 급락 종목을 자동으로 발굴하는 에이전트.

## 역할

- KOSPI/KOSDAQ/S&P500/NASDAQ100 유니버스에서 단기 급락 종목 스크리닝
- 낙폭 + 과매도 + 센티멘트 + 섹터 모멘텀 + 경쟁사 비교를 종합해 Top N 후보 선정
- 선정된 종목을 orchestrator로 전달하여 전체 분석 파이프라인 실행

## 스크리닝 기준

### 1단계: 낙폭 필터
- 3일 낙폭 ≥ -5% OR 10일 낙폭 ≥ -10%
- 20일 낙폭 ≥ -8%
- 52주 고점 대비 ≥ -20%

### 2단계: 과매도 조건 (1개 이상)
- RSI(14) < 35
- 볼린저밴드 하단 근접 (BB% < 0.20)
- Stochastic K < 25
- 거래량 급증 (20일 평균 ≥ 2배)

### 3단계: 종합 점수화
| 요소 | 가중치 |
|------|--------|
| 낙폭 크기/패턴 | 35% |
| 심리 지표 (RSI다이버전스, 거래량클라이맥스, OBV) | 25% |
| 섹터 모멘텀 (약세 섹터 = 반등 기대) | 20% |
| 동종업계 대비 밸류에이션 | 20% |

## 심리 지표

| 지표 | 해석 |
|------|------|
| RSI 상승 다이버전스 | 가격 저점 ↓ + RSI 저점 미달 → 반등 선행 |
| 매도 클라이맥스 | 거래량 2.5배+ + 당일 하락 → 단기 바닥 신호 |
| OBV 매집 | 가격 하락 중 OBV 상승 → 스마트머니 매집 |
| 극단적 공포 | RSI < 25 + MA 이탈 → 과도한 비관 |

## 실행 방법

```bash
python main.py --discover --market KR US --top-n 5
```

## 출력

- `output/screening_{date}.md` — 스크리닝 결과 마크다운
- 상위 종목은 orchestrator → idea-generator → reporter 파이프라인으로 전달

# peer-analyst

동종업계 경쟁사 밸류에이션 비교 분석 에이전트.

## 역할

- 대상 종목의 섹터/산업 정보를 기반으로 동종업계 피어 탐색
- PER/PBR/ROE를 동종업계 중앙값과 비교
- Claude API로 저평가/고평가 원인 및 경쟁 우위 분석 보고서 생성

## 분석 항목

| 지표 | 설명 |
|------|------|
| PER 할인율 | (대상 PER - 동종업계 중앙값) / 중앙값 × 100 |
| PBR 할인율 | 동일 방식 |
| ROE 비교 | 수익성 대비 저평가 여부 확인 |
| 1개월 상대 강도 | 대상 수익률 - 동종업계 평균 수익률 |

## 밸류에이션 판단 기준

- **저평가**: PER/PBR 평균 할인율 < -20%
- **적정**: -20% ~ +20%
- **고평가**: > +20%

## 피어 탐색 방법

- **US**: S&P500 내 동일 GICS Sector 종목
- **KR**: pykrx KOSPI 시총 상위 종목 (동일 업종 분류)

## 출력

- `output/peer_analysis_{ticker}_{date}.md` — 비교 분석 보고서
- idea-generator의 `peer_analysis` 파라미터로 전달되어 투자 아이디어에 반영
